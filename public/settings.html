Audio Track Setup

Multi-track audio requires SRT input. Configure two audio tracks in OBS so music is separated from the clean mix.

Track layout
Track 1
Full mix
Game + voice + music — sent live to Twitch viewers
Track 2
Clean mix (VOD-safe)
Game + voice only, no music — used for recordings, YouTube, Kick, TikTok, and Twitch VODs
OBS configuration steps
  1. Settings → Output — set Output Mode to Advanced
  2. Output → Streaming — set Audio Track to 1
  3. Settings → Audio — make sure at least 2 tracks are active (Tracks 1 and 2)
  4. Audio Mixer — click the gear icon on each audio source → Advanced Audio Settings:
    • Music sources → enable Track 1 only (excluded from clean mix)
    • Game, voice, alerts → enable Track 1 and Track 2

    Example: Music is assigned to Track 1 only. All other sources use both Track 1 and Track 2.

  5. Settings → Output → Audio — set Audio Encoder to AAC (not Opus) so copy mode works without transcoding
Per-platform audio routing (config.yaml)
twitchVodTrack: 2
Sends both tracks to Twitch + injects VOD metadata. Live viewers hear Track 1; Twitch saves Track 2 to the VOD archive.
vodOnly: true
Sends only Track 2 (clean mix). Music never reaches the platform. Use for YouTube, Kick, TikTok.
(no setting)
Sends only Track 1 (full mix). Use for platforms where VOD music licensing is not a concern.
recording
Always records Track 2 (clean mix) so saved files are VOD-safe.
